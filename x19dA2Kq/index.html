Setup Map

Setup Map shows where files go

This section explains installation. Use it when setting up Claude Projects, ChatGPT Projects, custom instructions, or lean context for DeepSeek, Gemini, and Perplexity.

Do first

Set up Claude Executive OS because Claude has Microsoft 365 and HubSpot context.

Do not do

Do not upload the entire zip into every model. That creates bloat and context drift.

Use loadouts

Model Chains

Use more than one model only when it adds value

A model chain is a deliberate handoff. One model retrieves context, another builds strategy, another challenges, another verifies sources, and another produces the asset. Use chains for GTM, pricing, CRM architecture, leadership updates, and external-facing material.

Use chains for

Pricing, GTM, product strategy, board-sensitive updates, major client comms, and public positioning.

Do not chain for

Simple rewrites, low-risk emails, formatting, or basic summaries. That is productivity theatre.

How to use

Run the first model, copy the output, then use the Next-model prompt from Prompt Builder.

Useful chains

Each chain reflects what each model is best at.

Strategy chain

ChatGPT builds. DeepSeek challenges. Perplexity verifies. ChatGPT integrates. Claude adapts internally.

Internal comms chain

Claude retrieves context. Claude drafts for stakeholder. ChatGPT sharpens. Stuart approves.

CRM chain

Claude retrieves HubSpot context. Claude translates into CRM mechanics. ChatGPT challenges architecture. Claude briefs Jo.

Asset chain

ChatGPT narrative. Gemini build. Claude internal fit. ChatGPT final polish.

Persistent Context

Persistent Context stores temporary working notes

This saves context in your browser only. Use it for current priorities, open loops, deadlines, or short-lived people notes that should appear in generated prompts. Do not use it for permanent knowledge. Permanent knowledge belongs in the markdown files and project loadouts.

Good use

Current GTM priorities, this week's CRM risks, live stakeholder sensitivities, and upcoming deadlines.

Bad use

Full stakeholder profiles, company strategy, pricing frameworks, or anything that should be maintained centrally.

How to inject

Save it here, then tick “Include persistent context” in Prompt Builder.

Saved live context

Stored locally in this browser. It will not sync across devices.
